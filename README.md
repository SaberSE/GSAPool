# GSAPool
Pytorch Implementation of GSAPool (WWW 2020)
====

PyTorch implementation of [Structure-Feature based Graph Self-adaptive Pooling](https://arxiv.org/pdf/2002.00848)




## Requirements
  * pytorch
  * torch_geometric

## Usage

```python control_shell.py```
